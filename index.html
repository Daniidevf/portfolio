Je suis Madani Traoré

Je suis un jeune Malien passionné par les nouvelles technologies depuis mon plus jeune âge. Ma fascination pour les mathématiques m'a naturellement conduit vers la programmation. Je suis une personne déterminée, sérieuse et adaptable, avec une forte capacité d'apprentissage.

Je parle couramment le Bambara, le Français et l'Anglais. Mes loisirs incluent la lecture, les jeux vidéo et le sport.

Avec plus de 3 ans d'expérience dans le développement web et mobile, j'ai travaillé sur divers projets allant des applications d'entreprise aux solutions IoT innovantes.

Description du Projet

Addinn

Plateforme e-commerce complète pour Addinn Paris, permettant la gestion des ventes, des utilisateiurs et des stocks via une interface intuitive.

Langages & Framework
Détails du Projet
  • Conception et développement d'une application web responsive
  • Systèm d'administration avancé pour la gestion des comptes
  • Tableau de bord analytique pour le suivi des performances
  • Intégration de paiements sécurisés via Stripe

Orange Foro Lakana

Description du Projet

Foro Lakana

Solution IoT combinant matériel embarqué et intelligence artificielle pour détecter et diagnostiquer les infections des plantes dans les exploitations agricoles.

Langages & Framework
Détails du Projet
  • Conception d'un dispositif IoT avec capteurs environnementaux
  • Développement d'algorithmes d'IA pour l'analyse d'images de plantes
  • Application mobile permettant aux agriculteurs de scanner leurs plantes
  • Plateforme web pour le suivi des exploitations agricoles
  • Système d'alertes et recommandations personnalisées
Résultats

La solution a permis de réduire les pertes agricoles de 35% dans les régions pilotes et a reçu le prix de l'innovation technologique au Mali.
